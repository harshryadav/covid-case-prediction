📦 INITIAL SETUP (One Time Only)
═══════════════════════════════════════════════════════════════════════════════

  1. Create environment:     python3 -m venv venv
  2. Activate it:            source venv/bin/activate
  3. Install packages:       pip install -r requirements.txt


🚀 QUICK START (Every Time)
═══════════════════════════════════════════════════════════════════════════════

  # Three simple commands:
  cd covid-case-prediction
  source venv/bin/activate
  python run_pipeline.py

  ⏱️  Takes: ~10-15 minutes
  📊 Output: results/ folder with plots and metrics


🔧 RUN INDIVIDUAL STEPS
═══════════════════════════════════════════════════════════════════════════════

  source venv/bin/activate                          # ALWAYS START WITH THIS!
  
  python src/data_processing/preprocess.py          # Download & clean data
  python src/data_processing/prepare_gluonts.py     # Prepare time series
  python src/visualization/create_eda_plots.py      # Create EDA plots
  python src/models/train_baseline.py               # Train naive models
  python src/models/train_deepar.py                 # Train neural network


📊 VIEW RESULTS
═══════════════════════════════════════════════════════════════════════════════

  open results/deepar_forecast.png                  # View forecast plot
  open results/                                     # View all results
  cat results/deepar_metrics.csv                    # View metrics


🐛 TROUBLESHOOTING
═══════════════════════════════════════════════════════════════════════════════

  Problem: "ModuleNotFoundError"
  Fix:     source venv/bin/activate
           pip install -r requirements.txt

  Problem: Pipeline fails
  Fix:     Run individual steps to find issue

  Problem: Out of memory
  Fix:     Edit src/models/train_deepar.py, set EPOCHS = 5
